## 🎚️ 智能延迟示例

```
场景: 用户在填充表单

传统延迟:
  ┌─────────────┐
  │ 0.5-2.0秒   │  ← 完全随机，接近固定
  └─────────────┘

新的自适应延迟:
  
  基础延迟 (0.5-2.0秒)
    │
    ├─ context="click" → 乘以点击延迟系数 (0.234)
    │
    ├─ context="type" → 使用打字速度 (0.087s/字)
    │
    ├─ context="read" → 乘以阅读时间系数 (1.2-3.0)
    │
    └─ context="think" → 随机长时间停顿 (1-4秒)
                        + 思考概率检查
```
